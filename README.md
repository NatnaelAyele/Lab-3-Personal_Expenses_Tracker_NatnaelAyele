 Personal Expense Tracker

A simple command-line personal expense tracker (Python) with a Bash script to archive expense files.

Description
- Track balance and expenses from the terminal.
- Save expenses to files named `expenses_YYYY-MM-DD.txt`.
- Archive old expense files into an `archives/` folder with a log.

Files
- `expenses-tracker.py` — main Python program
- `archive_expenses.sh` — Bash script to move expense files to `archives/` and log the action
- `balance.txt` — stores current balance and totals
